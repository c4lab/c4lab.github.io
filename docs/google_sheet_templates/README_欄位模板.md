# c4Lab Google Sheets 欄位模板（空白）

這份資料夾提供可直接匯入 Google Sheets 的空白 CSV 欄位模板。

## 重要程度標註

- `P0`：上線必備，請優先收齊
- `P1`：建議收齊，可強化內容品質與搜尋表現
- `P2`：選配，依實際需求決定

## 匯入方式（Google Sheets）

1. 建立一份新的 Google 試算表。
2. 依序以「檔案 -> 匯入 -> 上傳」匯入各 CSV。
3. 每次匯入選擇「插入新工作表」。
4. 工作表名稱建議直接使用 CSV 檔名。
